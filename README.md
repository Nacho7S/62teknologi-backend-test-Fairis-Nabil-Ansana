### Test Backend (Node Js) (Sequelize ORM)

## Installation 
```bash
npm run install
```

## Setup

config the database and run this script
```bash
npx sequelize db:create
```

migrate 
```bash
npx sequelize db:migrate
```

seeding data from api
```bash
npx sequelize db:seed:all
```

## Running Script 
running server (port: 3000)
```bash
npm run start
```
